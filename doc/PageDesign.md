Design a futuristic cyberpunk-style web interface inspired by Cyberpunk 2077.

Overall style:

* Dark theme with deep black and dark blue background
* Neon accents (cyan, purple, pink)
* Glow effects, glassmorphism panels, and subtle noise textures
* Futuristic UI similar to a hacker console or AI control system

Pages to design:

1. Home Page:

* Fullscreen hero section with large neon title “AI BLOG SYSTEM”
* Subtitle: “High Performance · Distributed · AI Powered”
* Two glowing buttons: “Enter System” and “View Architecture”
* Background with gradient, particles, and light streaks
* Below hero: blog post list in grid layout (2–3 columns)
* Each card: semi-transparent, glowing border, hover elevation effect

2. Blog Detail Page:

* Three-column layout:

  * Left: table of contents (sticky, neon highlight)
  * Center: markdown content (dark theme, code blocks with glow)
  * Right: info panel (AI summary, tags, metadata)
* Style like a futuristic terminal or hacker reading interface

3. Architecture Page:

* Title: “System Architecture”
* Central interactive diagram:

  * Nodes: Frontend, Gateway, Blog Service, AI Service, Redis, MySQL
  * Glowing connection lines with animated data flow
* Below: service cards with description and tech stack
* Overall feel: high-tech system dashboard

Design should feel immersive, high-tech, and slightly dystopian, with strong visual hierarchy and smooth hover animations.
